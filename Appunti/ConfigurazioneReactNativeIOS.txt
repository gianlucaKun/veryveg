Configurazione

Installa EXPO  (set di strumenti che aiuta a sviluppare app react Native senza necessita emulatore ios

# Install Expo CLI
npm install -g expo-cli

# Creare un nuovo progetto con Expo
expo init MyFoodScannerApp

# Vai nella directory del progetto
cd MyFoodScannerApp

# Avviare il progetto
npx expo start
